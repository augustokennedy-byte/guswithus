**For:** Arnau Sagrera-Barnet (info@velorfilms.com)

Welcome aboard, Arnau! Here's everything you need to manage the GUS web presence.

## What you have access to

**1. The website (GitHub)**
- The site lives in a GitHub repository; you'll get an invite by email — accept it,
  then you can edit from the repo page in any browser (no software needed).
- The whole site is one file: `index.html`. To edit text: open the file on
  github.com → click the pencil icon → make changes → "Commit changes".
  The live site updates automatically within about a minute.
- Every change is saved in history and reversible, so you can't break anything
  permanently. Images live in the `assets/` folder (Add file → Upload files).

**2. Donations (Zeffy)**
- You'll get a member invite to our Zeffy account by email.
- Zeffy is where donation forms, ticketing for events, and donor receipts live.
  You can create campaign forms, export donor lists, and see totals.
- Bank/payout settings stay admin-only (Gus).

## House rules
- Brand colors: green #22A95A and blue #0848A2 — they're set as variables at the
  top of `index.html` (`--green`, `--blue`), so never hard-code new colors.
- Tagline stays: "Where Potential Meets Purpose."
- Donation messaging always leads with: 100% of donations reach our programs
  (Zeffy charges us nothing).
- Big changes (new sections, redesigns): talk to Gus first.
